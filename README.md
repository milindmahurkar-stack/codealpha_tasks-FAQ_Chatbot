# 🤖 AI FAQ Chatbot with NLP-Based Question Matching

## 📌 Project Overview

AI FAQ Chatbot is a simple Artificial Intelligence project developed using Python and Streamlit. The chatbot answers frequently asked questions related to Artificial Intelligence and Technology using NLP-based question matching techniques.

Instead of exact keyword matching, the chatbot identifies the most similar question from its knowledge base and provides the appropriate answer.

## 🧠 How It Works

1. User enters a question.
2. The chatbot preprocesses the text.
3. RapidFuzz finds the most similar FAQ question.
4. The chatbot retrieves the best matching answer.
5. The response is displayed instantly.
